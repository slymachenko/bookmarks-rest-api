# bookmarks-rest-api

## Description

Bookmarks REST API created with [freeCodeCamp.org NestJs Course](https://www.youtube.com/watch?v=GHTA143_b-s&ab_channel=freeCodeCamp.org).

## Installation

```bash
$ npm install
```

Set up env vars in [.env](/.env.example) and [.env.test](/.env.test.example)

## Running the app

Start Docker Daemon

```bash
$ npm run db:dev:up
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
